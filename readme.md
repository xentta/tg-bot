Методы Markup.keyboard
1. .resize()
   Описание: Делает клавиатуру компактной, подгоняя её под количество кнопок. Это предотвращает использование большого пространства на экране.
   По умолчанию: Клавиатура занимает всё доступное место.
   Пример:
   javascript
   Копировать код
   Markup.keyboard([
   ['Кнопка 1', 'Кнопка 2']
   ]).resize();
2. .oneTime()
   Описание: Указывает, что клавиатура должна исчезнуть после нажатия любой кнопки. Это полезно для действий, где клавиатура нужна только один раз.
   Пример:
   javascript
   Копировать код
   Markup.keyboard([
   ['Кнопка 1', 'Кнопка 2']
   ]).oneTime();
3. .selective()
   Описание: Показывает клавиатуру только определённым пользователям, вместо всех участников чата. Полезно в групповых чатах, чтобы клавиатура была видима только инициатору команды.
   Пример:
   javascript
   Копировать код
   Markup.keyboard([
   ['Кнопка 1', 'Кнопка 2']
   ]).selective();
4. .removeKeyboard()
   Описание: Удаляет клавиатуру из интерфейса пользователя.
   Пример:
   javascript
   Копировать код
   Markup.removeKeyboard();
5. .placeholder(text)
   Описание: Устанавливает текст-заполнитель (placeholder) в поле ввода текста, пока клавиатура активна. Это может быть полезно для указания инструкций.
   Пример:
   javascript
   Копировать код
   Markup.keyboard([
   ['Кнопка 1', 'Кнопка 2']
   ]).resize().placeholder('Введите ваш текст...');
6. .keyboard(buttons)
   Описание: Основной метод для создания самой клавиатуры. Принимает массив кнопок и возвращает объект клавиатуры.
   Пример:
   javascript
   Копировать код
   Markup.keyboard([
   ['Кнопка 1', 'Кнопка 2'],
   ['Кнопка 3']
   ]);
